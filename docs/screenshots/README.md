# Screenshot Inventory

Save screenshots here before documenting the release. Use PNG unless there is a
specific reason to use another format.

Recommended filenames:

- `01-plugin-menu.png` - QGIS plugin menu with settings, load/save, and export actions.
- `02-dock-basic.png` - Kakao Map / Roadview dock opened inside QGIS.
- `03-position-sync.png` - QGIS canvas, Kakao Map, and Roadview synchronized.
- `04-local-search.png` - Place/address search results and selected result.
- `04-local-search_search_input.png` - Search input with result list opened.
- `05-roadview-layer.png` - `Kakao Roadview Position` layer and direction marker.
- `06-route-input.png` - Route origin, destination, waypoint, avoid, and vehicle controls.
- `07-route-result.png` - Generated route, route points, and guidance layers in QGIS.
- `08-guidance-selection.png` - Guidance item selected and focused on the map.
- `09-route-history.png` - Route history tab with actions.
- `10-external-browser.png` - External browser integration mode.
- `11-export-files.png` - Optional export result files or reloaded export layers.
- `12-gpx-load-menu.png` - GPX styled load menu action.
- `12-gpx-load-output.png` - GPX load result with QML styles applied.
- `13-layout-map-left.png` - Map/Roadview left-right layout.
- `14-fullscreen.png` - Fullscreen toggle example.
- `15-Plugin-menu_javascript_api_key.png` - JavaScript API key menu item.
- `15-Plugin-menu_rest_api_key.png` - REST API key menu item.

Before committing screenshots:

- Hide or blur API keys.
- Hide personal account information.
- Avoid exposing sensitive home/work locations.
- Keep the QGIS Layers panel visible when the screenshot proves generated layers.
- Prefer a consistent window size for related screenshots.
